安裝 Express

參考 Reference 2，依照下列步驟進行安裝 Express 4.17.1

1. Express 需要跟隨在應用程式上，所以需要先建立應用程式資料夾

2. 使用 npm init 指令建立 package.json 檔案。請參考 Specifics of npm’s package.json handling 了解 package.json 運作細節

3. 建立過程會有許多資訊需要輸入

4. 開始安裝 express ，並使用 --save指令 to save it in the dependencies list.
