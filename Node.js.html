Node.js & Express Study Note

安裝 Node.js

參考 Reference 1，依照下列步驟安裝 Node.js 12.x。

1. 安裝 Node.js

''' curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash - sudo apt-get install -y nodejs '''

2. You may also need development tools to build native addons.

''' sudo apt-get install -y build-essential gcc g++ make '''

3. 安裝 Yarn Package Manager

''' curl -sL https://dl.yarnpkg.com/debian/pubkey.gpg | sudo apt-key add - echo "deb https://dl.yarnpkg.com/debian/ stable main" | sudo tee /etc/apt/sources.list.d/yarn.list sudo apt-get update && sudo apt-get install yarn '''

安裝 Express

參考 Reference 2，依照下列步驟進行安裝 Express 4.17.1

1. Express 需要跟隨在應用程式上，所以需要先建立應用程式資料夾

''' mkdir myapp cd myapp '''

2. 使用 npm init 指令建立 package.json 檔案。請參考 Specifics of npm’s package.json handling 了解 package.json 運作細節

''' npm init '''

3. 開始安裝 express ，並使用 --save指令 to save it in the dependencies list.

''' npm install express --save '''
